# Piadas do Chuck Norris

Melhorias aplicadas neste repositório:

- Separação dos estilos em `styles.css` (identidade visual profissional).
- Paleta: preto, branco, vermelho, azul e cinzento.
- Correção do fetch da API para `https://api.chucknorris.io/jokes/random`.
- Melhorias de acessibilidade (`role="status"`, `aria-live`, foco visível).
- Layout responsivo e botões com estados (hover, foco, disabled). 

Como abrir:

1. Abra o arquivo [index.html](index.html) no seu navegador (duplo clique) ou sirva com um servidor estático.

Arquivos principais:

- [index.html](index.html)
- [styles.css](styles.css)

Sugestões futuras:

- Subir em GitHub Pages ou Netlify.
- Adicionar fonte customizada (Ex: Inter via Google Fonts) para maior refinamento.
